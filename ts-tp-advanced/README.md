# TP TypeScript — Solutions fournies

Structure du projet:
- src/ : code source TypeScript
- dist/ : code compilé (généré après `npm run build`)
- package.json, tsconfig.json, .eslintrc.json, .prettierrc

Commandes:
```bash
# installer les dépendances (localement si souhaité)
npm install

# compiler TypeScript
npm run build

# exécuter
npm run start

# vérifier linting (si eslint installé)
npm run lint

# formater
npm run format
```

Chaque exercice est dans `src/exercises` et il y a un mini-projet "Library" dans `src/library`.
